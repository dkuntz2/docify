
Get a factorial *recursively*

Standard factorial method using recursion, returns 1 if x is less than
or equal to 1, otherwise it returns `factorial(x-1)`.

## Parameters

### x 

 integer to the the factorial of.
		

## Returns

### defalut 

 x!
